link

تگ <link> در HTML برای اتصال یک فایل خارجی به سند HTML استفاده می‌شود. این تگ معمولاً در بخش head صفحه قرار می‌گیرد و برای پیوند دادن فایل‌های CSS، آیکون‌ها یا سایر منابع خارجی استفاده می‌شود.

ساختار تگ <link>

تگ <link> یک تگ خودبسته (self-closing) است و نیازی به تگ بسته ندارد. ساختار کلی آن به صورت زیر است:

ویژگی‌های اصلی تگ <link>

rel: مشخص می‌کند نوع رابطه سند HTML با فایل خارجی چیست (مثلاً "stylesheet" برای CSS).

href: آدرس فایل خارجی که می‌خواهید پیوند دهید.

اتصال فایل CSS خارجی

برای پیوند دادن یک فایل CSS به صفحه HTML:

rel="stylesheet": مشخص می‌کند فایل خارجی یک فایل استایل است.

href="styles.css": آدرس فایل CSS.
