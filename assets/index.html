HTML

(HyperText Markup Language)

یک زبان نشانه‌گذاری استاندارد برای ایجاد و طراحی صفحات وب است.این زبان به مرورگرهای وب کمک می‌کند تا ساختار و محتوای صفحات وب را تفسیر و نمایش دهند. HTML از تگ‌ها و عناصر مختلفی استفاده می‌کند تا بخش‌های گوناگون یک صفحه مانند متن، تصاویر، لینک‌ها، جدول‌ها و فرم‌ها را مشخص کند.
